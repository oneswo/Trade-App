## 1. 项目概述

面向海外买家的二手工程机械外贸独立站，包含前台展示和后台管理系统。付费投流引流模式。

| 项目 | 说明 |
|------|------|
| 业务类型 | 二手工程机械出口（挖掘机、装载机、推土机等） |
| 目标用户 | 海外买家（非洲、中东、南美、东南亚等） |
| 变现模式 | 询盘留资 → 人工跟单成交 |
| 核心功能 | 产品展示、询盘留言、多语言、管理后台 |
| 部署目标 | Vercel + Cloudflare R2 + 真实数据库 |

## 2. 架构概览

| 组件 | 服务商 | 职责 | 状态 |

| 本地开发文件存储 | 本地 `public/uploads/` | 开发阶段临时方案 | ✅ 已实现 |
| 本地开发数据库 | 内存 Mock Store | 开发阶段临时方案 | ✅ 已实现 |

> **本地开发 → 上线切换机制：** 所有基础设施通过环境变量自动切换，`DATABASE_URL` 存在时用 Supabase，`R2_ACCOUNT_ID` 存在时用 R2，否则继续用本地 mock，**不需要改任何代码**。

### Supabase 选型说明

| 项目 | 说明 |
|------|------|
| 数据库类型 | PostgreSQL（真正的关系型数据库） |
| 推荐区域 | 新加坡（ap-southeast-1），延迟约 80-120ms |
| 免费额度 | 500MB 数据库 + 1GB 文件存储，初期够用 |
| 管理界面 | 自带网页控制台，可直接查看/修改数据 |
| Vercel 集成 | 官方支持一键集成，环境变量自动注入 |
| SDK | @supabase/supabase-js 2.x，已安装 |

> ⚠️ **免费套餐注意：** 超过 7 天不活跃会自动暂停项目，需手动唤醒。正式上线后建议升级到 Pro 套餐（$25/月），或上线初期每周登录一次 Supabase 控制台保活。

## 3. 流量链路

```
海外用户 → 域名 → Vercel 全球边缘节点 → Next.js 应用
产品图片/视频 → Cloudflare R2（公开访问）
询盘表单提交 → Next.js API Routes → 数据库
管理后台 → /admin/* → Next.js API Routes → 数据库
```

> 国内加速备选：如需国内访问加速，可参考 EngVlog 方案接入阿里云 ESA（NS 接入 + 回源 Vercel），无需改动代码。

| 语言 | 代码 | 说明 |
|------|------|------|
| 英语 | `en` | 主语言，默认 |

> 扩展说明：新增语言只需在 `/src/messages/` 目录添加对应翻译 JSON 文件，next-intl 路由自动支持，无需改动代码。

### 5.2 路由结构

```
/en/              英语首页
/zh/products      中文产品列表
/ar/about         阿拉伯语关于页（RTL）
/admin/*          管理后台（不走多语言路由）
```

### 5.3 RTL 支持

阿拉伯语需要 RTL（从右到左）布局，在 `<html>` 标签动态注入 `dir` 属性即可，Tailwind CSS 4 内置支持。

## 6. 文件存储

### 6.1 本地开发（当前）

上传文件保存至 `public/uploads/{image|video}/{YYYYMMDD}/{timestamp}-{uuid}.{ext}`，通过相对路径 `/uploads/...` 直接访问。

### 6.2 生产环境（待接入 R2）

实现 `R2MediaStorage` 类（接口与本地存储完全相同），`R2_ACCOUNT_ID` 环境变量存在时自动切换，代码不变。

### 6.3 存储路径规范（R2）

| 文件类型 | 存储路径 |
|----------|---------|
| 产品封面图 | `/products/{productId}/cover.{ext}` |
| 产品相册图 | `/products/{productId}/gallery/{n}.{ext}` |
| 产品视频 | `/products/{productId}/video.{ext}` |
| 文章封面图 | `/articles/{articleId}/cover.{ext}` |

### 6.4 删除联动

删除产品或文章时，同步删除 R2 中对应的所有文件（封面图、相册图、视频），防止废弃文件堆积。
