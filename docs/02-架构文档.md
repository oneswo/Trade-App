# 外贸独立站架构文档

技术架构 · 配置说明 · 交接提示词  
文档更新时间：2026年3月27日

---

## 1. 项目概述

外贸独立站，面向海外买家，主要功能为产品展示（图片+视频+文字）、多语言切换、用户询价留言管理。付费投流模式，暂不考虑 SEO。

| 项目 | 说明 |
|------|------|
| 目标用户 | 海外买家为主 |
| 变现模式 | 付费投流引流 |
| 核心功能 | 产品展示、询价留言、多语言、管理后台 |
| UI 风格 | 工业品牌站风格，高级感，优于对标网站 |

---

## 2. 架构概览

Vercel 负责代码部署，Cloudflare R2 负责图片/视频文件存储，Supabase 负责数据库，多语言使用 next-intl。如后续有国内用户需求，可接入阿里云 ESA 加速（参考 EngVlog 方案）。

| 组件 | 服务商 | 职责 | 版本/技术 |
|------|--------|------|----------|
| 代码部署 | Vercel | Next.js 部署，自动 CI/CD，全球 CDN | Next.js 15+, React 19 |
| 文件存储 | Cloudflare R2 | 产品图片、视频文件 | S3-compatible API |
| 数据库 | Supabase | 产品数据、询价留言、后台账号 | PostgreSQL |
| 多语言 | next-intl | 国际化路由与翻译 | next-intl 最新版 |
| 前端框架 | - | UI 组件与样式 | Tailwind CSS 4, Radix UI |
| 状态管理 | - | 全局状态 | Zustand |
| 表单验证 | - | 询价表单校验 | Zod |
| 富文本编辑器 | - | 页面内容编辑 | TipTap |

---

## 3. 流量链路

```
海外用户 → 域名 → Vercel 全球边缘节点（自动就近）→ Next.js 应用
产品图片/视频 → Cloudflare R2（公开访问）
询价表单提交 → Next.js API Routes → Supabase
管理后台 → /admin/* → Next.js API Routes → Supabase
```

> 国内加速备选：如需国内访问加速，参考 EngVlog 方案接入阿里云 ESA（NS 接入 + 回源 Vercel），无需改动代码。

## 5. 多语言方案

### 5.1 支持语言（第一期）

| 语言 | 代码 | 说明 |
|------|------|------|
| 英语 | `en` | 主语言，默认 |
| 中文 | `zh` | 管理方便，兼顾国内 |
| 阿拉伯语 | `ar` | 中东市场，RTL 布局 |
| 俄语 | `ru` | 俄语市场 |
| 西班牙语 | `es` | 拉美/欧洲市场 |

> 扩展说明：新增语言只需在 `/messages/` 目录添加对应的翻译 JSON 文件，next-intl 路由自动支持，无需改动代码逻辑。

### 5.2 路由结构

```
/en/products        英语产品列表
/ar/products        阿拉伯语产品列表
/ru/products        俄语产品列表
/es/products        西班牙语产品列表
```

### 5.3 RTL 支持

阿拉伯语需要 RTL（从右到左）布局，Tailwind CSS 内置 `dir="rtl"` 支持，在 `<html>` 标签动态注入 `dir` 属性即可。

## 6. 文件存储（Cloudflare R2）

| 配置项 | 值 |
|--------|-----|
| 存储桶 | Cloudflare R2 |
| API 类型 | S3-compatible (AWS SDK v3) |
| 文件类型 | 产品图片、产品视频 |

### 6.1 存储路径规范

| 文件类型 | 存储路径示例 |
|----------|-------------|
| 产品封面图 | `/products/{productId}/cover.jpg` |
| 产品图片组 | `/products/{productId}/images/{n}.jpg` |
| 产品视频 | `/products/{productId}/video.mp4` |
| 分类图标 | `/categories/{categorySlug}/icon.png` |

## 7. 数据库结构（Supabase）

### 7.1 products 表（产品）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键 |
| slug | text | URL 友好名称，唯一 |
| category_id | uuid | 外键，关联分类表 |
| cover_url | text | 封面图地址 |
| images | text[] | 图片组地址数组 |
| video_url | text | 视频地址（可为空） |
| is_published | boolean | 是否上架 |
| created_at | timestamp | 创建时间 |
| updated_at | timestamp | 更新时间 |

> 多语言内容（标题、描述）存储在独立的 `product_translations` 表。

### 7.2 product_translations 表（产品多语言内容）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键 |
| product_id | uuid | 外键，关联产品表 |
| locale | text | 语言代码：en/zh/ar/ru/es |
| title | text | 产品标题 |
| description | text | 产品描述 |

### 7.3 categories 表（产品分类）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键 |
| slug | text | URL 友好名称，唯一 |
| icon_url | text | 分类图标 |
| sort_order | integer | 排序权重 |
| is_active | boolean | 是否启用 |

### 7.4 category_translations 表（分类多语言）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键 |
| category_id | uuid | 外键 |
| locale | text | 语言代码 |
| name | text | 分类名称 |

### 7.5 inquiries 表（询价留言）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键 |
| name | text | 姓名 |
| email | text | 邮箱 |
| phone | text | 电话（可为空） |
| message | text | 询价内容 |
| product_id | uuid | 关联产品（可为空，通用询价） |
| locale | text | 用户使用的语言 |
| is_read | boolean | 是否已读 |
| created_at | timestamp | 提交时间 |

### 7.6 pages 表（页面主表）

关于我们、服务介绍等图文页面的标识，不存内容，只存页面 slug。

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键 |
| slug | text | 页面标识：about / service / news 等，唯一 |
| is_active | boolean | 是否启用 |
| created_at | timestamp | 创建时间 |
| updated_at | timestamp | 更新时间 |

### 7.7 page_translations 表（页面多语言内容）

每个页面的各语言版本内容，富文本使用 TipTap JSON 格式存储。

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键 |
| page_id | uuid | 外键，关联 pages 表 |
| locale | text | 语言代码：en/zh/ar/ru/es |
| title | text | 页面标题 |
| content | text | 富文本内容（TipTap JSON 序列化字符串） |
| updated_at | timestamp | 最后编辑时间 |

> 说明：后台编辑器保存的是 TipTap JSON，前台渲染时用 TipTap 的只读模式输出 HTML，不需要额外处理。

### 7.8 admins 表（管理员）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键 |
| email | text | 登录邮箱 |
| password_hash | text | 加密密码 |
| created_at | timestamp | 创建时间 |

## 8. 页面结构

### 8.1 前台页面

```
/[locale]/                      首页（Hero + 产品分类 + 精选产品 + 关于我们 + 联系）
/[locale]/products              产品列表（支持分类筛选）
/[locale]/products/[slug]       产品详情（图片轮播 + 视频 + 描述 + 询价按钮）
/[locale]/contact               联系我们 / 询价页
/[locale]/about                 关于我们
```

### 8.2 管理后台页面

```
/admin/login                    管理员登录
/admin/dashboard                数据概览（产品数、询价数、未读数）
/admin/products                 产品列表（增删改查）
/admin/products/new             新建产品
/admin/products/[id]/edit       编辑产品
/admin/categories               分类管理（增删改排序）
/admin/inquiries                询价留言列表（标记已读、查看详情）
/admin/pages                    页面内容管理（富文本编辑器）
```

## 9. 视频播放方案

产品视频使用原生 `<video>` 标签播放 MP4，两种展示方式：

| 方式 | 场景 | 实现 |
|------|------|------|
| 弹窗播放 | 产品列表页点击 | Radix UI Dialog + `<video>` |
| 内嵌播放 | 产品详情页 | 页面内 `<video>` 组件 |

> 具体采用哪种方式根据实际 UI 效果决定，代码结构两种都支持。
