# UI设计与页面架构文档 (前台 + 后台)

基于您的严格要求：**“高端、大气、定制化无AI感”、“绝对的对齐与呼吸感（强迫症友好）”**，特制定以下 UI 设计系统与整体页面架构规划。

## 第一部分：核心设计系统 (UI Design System)

### 1. 栅格与空间留白 (Whitespace & Grid)
- **基本单位**：全部遵循 **4px/8px** 基准。Margin/Padding 只允许使用 `16px, 24px, 32px, 48px, 64px, 96px, 128px`。
- **极简呼吸感**：在不同 Section（内容区块）之间使用超大间距 (128px 或 160px)，彻底消灭信息的拥挤感。
- **绝对对齐**：不论是左对齐还是居中对齐，全站组件的边界必须严丝合缝地处于同一条隐形垂直基准线上。所有图片强制采用黄金比例 (如 16:9 或统一方图)，防止错落不齐。

### 2. 精密排版 (Typography)
- **字体选择**：
  - **主修标题**：采用现代且富有工业张力的无衬线字体（推荐 `Inter` 或 `Outfit`），使用 `ExtraBold` 或 `Black` 的大字重。
  - **正文与参数**：采用纤弱优雅的 `Regular` (字重 400)，并增加绝佳的行高 (`leading-relaxed` 或 `1.75`)。
- **字间距技巧**：针对大号的大写英文字段 (如 ABOUT US)，适度拉开字间距 (`tracking-widest`)，极大地拉升国际高端感。

### 3. 克制的色彩与质感 (Color & Textures)
- **背景**：摒弃纯白 (`#FFFFFF`)，使用极其微弱的高级珍珠灰 (`#FAFAFA` 或 `#F8F9FA`) 作为大背景。
- **主色彩**：深邃的玄武黑 (`#111111`) 作为主文字与标题色。品牌点缀色仅限于 Button Hover 和重要图标，绝不大面积涂抹。
- **组件质感**：
  - 毛玻璃效果 (Glassmorphism)：用于吸顶导航栏、后台管理悬浮框。
  - 边框：杜绝所有粗重的黑边框。卡片分割使用微弱的底侧单线 (`border-b border-gray-100`) 或超大圆角与极其微弱的大弥散阴影 (`shadow-sm` 到 `shadow-2xl`，透明度压低到 3%)。

### 4. 优雅的微动效 (Micro-interactions)
- 所有 `hover`、`focus` 反馈强制加入 `transition-all duration-300 ease-in-out`，使一切操作如同顶级跑车的阻尼感般丝滑，杜绝廉价的闪跳。

## 第二部分：前台页面架构规划 (Frontend Structure)

前台承担着“品牌展示”与“客户获客(B2B Inquiry)”的重任。

1. **首页 (Home)**
   - 全局毛玻璃吸顶 Header。
   - 极具冲击力的大屏 Hero Area (建议后期采用电影级质感的减速视频背景)。
   - 极其克制、并排对齐的核心品牌优势展示。
   - 完美网格切割的 Featured Equipment 橱窗。
   - Footer：几何级平衡的底栏布局。
2. **产品目录 (Catalog / Equipment)**
   - 左侧：极简灰度化的树状折叠侧边栏过滤器。
   - 右侧：统一尺寸比例的产品卡片瀑布/网格。
3. **产品详情页 (Product Detail) —— 核心枢纽**
   - 上半部分：左侧极简粘性画廊 (Sticky Image Gallery) + 右侧参数速览与硕大圆润的 "Get Quote" 按钮。
   - 下半部分：取消传统 HTML Table，改为类似苹果官网技术规格参数一样的隐形分隔线 Data 列表。
   - 底部：全宽表单设计的询盘入口 (Inquiry Form)，极致的单行下划线输入框设计。
4. **其他单页 (About, Contact, News)**
   - 采用大留白的图文交叉布局。

## 第三部分：后台页面架构规划 (Admin Dashboard Structure)

高质量的外贸需要一个同样“高端、大气”的管理后台。我们将使用极简黑白灰风格，彻底抛弃传统的臃肿 Admin 模板。

### 1. 后台核心设计理念
- **侧边栏 (Sidebar)**：通体深空灰或纯净白，微弱图标，极简文字。选中项带有平滑的左侧高亮细线。
- **内容区与表格**：完全消除表格的纵向全包围边框 (No Vertical Borders)。行与行之间仅留微弱底线，表头全大写灰字，极具呼吸感。
- **弹窗与抽屉 (Modal & Drawer)**：放弃难看的行内编辑，所有的新增/修改产品全部使用从右侧滑出的宽体 Drawer 或者优雅居中带有背部模糊特效的 Modal。

### 2. 后台板块划分
1. **数据看板 (Dashboard)**
   - 顶部概览数据卡片 (Total Inquiries, Products Online, Weekly Views)，辅以细线极简图表。
   - 最新未读询盘滚动列表。
2. **商品管理 (Product Management)**
   - **分类管理 (Categories)**：支持无限级联分类，拖拽排序。
   - **产品列表 (Products Master)**：大图片缩略、一键上下架开关组件 (Switch component)。
   - **产品发布/编辑**：多图上传、规格参数动态添加（Key-Value 无限增减组）、富文本编辑器。
3. **获客管理 (Inquiry / Lead Management)**
   - 独立的消息列表栏：显示来自哪个产品、客户国家、时间轴。
   - 询盘详情抽屉：点击任一询盘，右侧丝滑拉出，展示详细的客户诉求与联系方式，并标记处理状态 (Pending, Contacted, Closed)。
4. **全站管理 (Site Setup)**
   - **轮播图与案例设置 (Banners & Cases)**
   - **内容管理 (News/Blog)**：SEO 友好文章发布引擎。
   - **全局配置 (Global Settings)**：公司地址、联系邮箱、WhatsApp 账号、社交链接、Logo 替换。
